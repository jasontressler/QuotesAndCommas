# Quotes and Commas

*ADS extension*

Adds a command palette command and context menu item "Add quotes and commas..."
that converts one or more lines of selected text into a stringified, comma-separated list.

**For example:**

*apple*
*banana*
*cat*
*dog*

**becomes**

*'apple',*
*'banana',*
*'cat',*
*'dog'*
